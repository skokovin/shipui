# Angular UI and Window handler for  [PutOut project](https://github.com/skokovin/putout)

Web App based on regular Angular framework. It loads, runs, and manages WebAssembly module compiled by RUST.

![Untitled video - Made with Clipchamp](https://github.com/skokovin/putout/assets/13080037/514d2e2b-2522-4032-9699-3133bf80d14b)


## Demos

Video instruction [YouTube](https://www.youtube.com/watch?v=E0fKqEAThts). </br>

[500mb ship](https://viewer701-f462d.web.app/). </br>


## Features

- Hide/Selct
- 6 sliders
- Measuring
- ID for DB requests
- Transparency
- Reset center of rotation
- Orbit/FPS cameras
- Primitive Snap
